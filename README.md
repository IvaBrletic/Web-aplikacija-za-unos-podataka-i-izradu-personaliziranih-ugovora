# Web aplikacija za unos podataka i izradu personaliziranih ugovora

## Opis

Ova web aplikacija izrađena je kao završni rad.

Omogućuje:

- unos podataka o tvrtki
- unos kontakt osobe
- unos stvarnog vlasnika
- unos dodatnih vlasnika
- unos prodajnih mjesta
- generiranje PDF ugovora
- administratorski pregled podataka
- uređivanje i brisanje korisnika
- pretraživanje korisnika prema OIB-u

## Tehnologije

- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja2

## Pokretanje

1. Instalirati Python
2. Instalirati biblioteke

pip install -r requirements.txt

3. Pokrenuti

python app.py

4. Otvoriti

http://127.0.0.1:5000
